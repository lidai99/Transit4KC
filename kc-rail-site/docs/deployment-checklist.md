# Deployment checklist

- [ ] Buy or configure a domain.
- [ ] Choose Node-capable hosting for the backend.
- [ ] Create a PostgreSQL database.
- [ ] Add environment variables from `.env.example`.
- [ ] Confirm `/` redirects to `/waldo-extension/`.
- [ ] Submit a test support form entry.
- [ ] Confirm the row appears in `support_feedback`.
- [ ] Review privacy language before public launch.
